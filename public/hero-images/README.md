# Hero Images Folder

Place your hero section images in this folder.

## Recommended Specifications:
- **Format**: JPG or PNG
- **Aspect Ratio**: 16:9 or 21:9 (wide format works best)
- **Resolution**: 1920x1080 or higher
- **File Size**: Optimize images to keep page load fast (aim for < 500KB per image)
- **Naming**: Use descriptive names like `hero-1.jpg`, `hero-2.jpg`, etc.

## Supported Formats:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

The images will automatically be displayed in the hero section with smooth fade transitions.
